자바스크립트 코드 입력 시 주의사항(규칙)

1. 자바스크립트는 대/소문자를 구분하여 작성한다.
ex) 날짜 객체 생성 : New date(); (X)
    날짜 객체 생성 : new Date(); (O)

2. 코드 한 줄을 작성한 후에는 세미콜론(;)으로 문장을 마무리한다.
   세미콜론을 사용하지 않으면, 한 줄에 2개 이상의 코드를 작성할 경우 오류가 발생한다.
ex) document.write("hi") document.write("bye") (X, 오류 발생)
    document.write("hi"); document.write("bye"); (O, 정상 작동)

3. 하지만 2와 같이 한 줄에 2개 이상의 코드를 작성하지 않고,
   한 줄에는 한 문장만 작성하는 것을 원칙으로한다.

4. 문자형 데이터를 작성할 때는 큰 따옴표("")와 작은따옴표('')를 겹치지 않게 주의한다.
ex) document.write("책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와있다.");
    -> 큰 따옴표가 한 문자열에 두번 사용되어 오류가 발생한다.

cf) 문자형 데이터를 작성할 때는 큰 따옴표 속에 작은 따옴표를 사용하거나,
    작은 따옴표 속에 큰 따옴표를 사용한다. 아니면 역슬래시를 이용하여 이스케이프 문자로 표현하는 방법도 있다.
cf ex) document.write('책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와있다.');
       or document.write('책에 \"자바스크립트는 대소문자를 구분해야 합니다\"라고 나와있다.');
       -> 두 문장 모두 정상 작동한다.

5. 코드를 작성할 때 중괄호{} 또는 소괄호()의 짝이 맞아야 한다.
ex) document.write("hi"; (X)
    document.write("hi"); (O)

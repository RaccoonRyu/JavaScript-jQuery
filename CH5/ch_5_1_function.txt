함수(function) 개요
- 우리는 앞에서 데이터를 저장할 때 변수를 선언해서 저장했다.
- 변수에는 데이터만 저장할 수 있고, 코드를 저장할 수는 없다.
- 하지만 함수를 사용하면 코드를 메모리에 저장했다가 필요할 때마다 호출하여 사용할 수 있다.

변수와 함수의 차이점
- 변수
1) 1개의 데이터만 저장
2) var라는 키워드를 이용하여 선언
3) 문자형, 숫자형, 논리형 데이터를 저장
4) 객체를 참조

- 함수
1) 자바스크립트 코드를 저장
2) function이라는 키워드를 이용하여 선언
3) 출력문, 제어문 등의 코드를 저장하고 데이터를 반환

기본 함수 정의문
- 함수 정의문 : 함수를 사용하여 코드를 저장한 것
- 변수를 선언할 때 var 키워드를 사용한 것 처럼 함수에서는 function 키워드를 사용해 변수를 선언한다
- 기본형 : function 함수명() { 자바스크립트 코드; }
- 익명함수(함수명이 없는 함수) 선언 및 변수 참조 : 참조변수 = function() { 자바스크립트 코드; }
- 함수 정의문에 작성된 코드는 즉시 실행되지 않고, 메모리에 할당되어 대기하고 있다가 함수가 호출되면 실행된다.
- 정의되어 있는 함수를 호출하는 기본형은 함수명(); / 참조변수();

호이스팅(hoisting)
- 호이스팅이란, 함수 정의문 보다 호출문이 먼저 나와도 함수 정의문을 끌어올려 함수를 호출하는 것이다.
- 일반 함수 정의(함수 정의문)는 함수 호출 시 호이스팅 기술을 지원한다.
- 하지만 익명 함수 선언 참조 방식은 호이스팅을 지원하지 않는다.

브라우저 객체 중 window 객체를 제외한 나머지 객체 일람

1. screen 객체
- 사용자의 모니터 정보(속성)를 제공하는 객체
- 기본형 : screen.속성;
- width 속성 : 화면의 너빗값을 반환
- height 속성 : 화면의 높잇값을 반환
- availWidth 속성 : 작업 표시줄을 제외한 화면의 너빗값을 반환
- availHeight 속성 : 작업 표시줄을 제외한 화면의 너빗값을 반환
- colorDepth 속성 : 사용자 모니터가 표현 가능한 컬러 bit를 반환

2. location 객체
- 사용자 브라우저와 관련된 속성, 메서드를 제공하는 객체
- 현재 URL에 대한 정보(속성)와 새로고침(reload) 메서드를 제공
- 기본형 : location.속성; / location.메서드();
- href 속성 : 주소 영역의 참조 주소를 설정하거나 URL을 반환 -> 사용자 브라우저의 URL 값을 가져오거나 URL 경로를 지정한 주소로 변경할 때 사용
- hash 속성 : URL의 해시값(#에 명시된 값)을 반환
- hostname 속성 : URL의 호스트 이름을 설정하거나 반환
- host 속성 : URL의 호스트 이름과 포트 번호를 반환
- protocol 속성 : URL의 프로토콜을 반환
- search 속성 : URL의 쿼리(요청값)를 반환
- reload() 메서드 : 브라우저를 새로고침(F5키 처럼)

3. history 객체
- 사용자가 방문한 사이트의 기록을 남기고, 이전 방문 사이트와 다음 방문 사이트로 다시 돌아갈 수 있는 속성과 메서드를 제공하는 객체
- 기본형 : history.속성; / history.메서드(); / history.메서드(숫자);
- back(이동 숫자) 메서드 : 이전 방문 사이트로 이동 (이동 숫자 입력시 입력한 숫자 만큼의 이전 방문 사이트로 이동)
- forward() 메서드 : 다음 방문 사이트로 이동
- go(이동 숫자) 메서드 : 입력한 이동 숫자 만큼의 방문사이트로 이동 (음수 : 이전 사이트로 이동, 양수 : 다음 사이트로 이동)
- length 속성 : 방문 기록에 저장된 목록의 개수를 반환

4. navigator 객체
- 현재 방문자가 사용하는 브라우저 정보와 운영체제 정보를 제공하는 객체
- 기본형 : navigator.속성;
- appCodeName 속성 : 현재 브라우저의 코드명 반환 (현 시점의 모든 브라우저는 Mozila를 반환한다.)
- appName 속성 : 현재 브라우저의 이름을 반환 (현 시점의 모든 브라우저는 Netscape를 반환한다.)
- appVersion 속성 : 현재 브라우저의 버전 정보를 반환 (현 시점의 모든 브라우저는 5.0(Windows)를 반환한다.)
- language 속성 : 현재 브라우저가 사용하고 있는 언어를 반환 (한국어를 사용할 경우 ko를 반환한다.)
- product 속성 : 현재 브라우저의 엔진 이름을 반환 (크롬의 경우 Gecko를 반환)
- platform 속성 : 현재 컴퓨터의 운영체제 정보를 제공
                (운영체제가 윈도우이고 시스템 종류가 64bit라도 브라우저가 32bit로 설치되었다면 Win32로 표시)
- onLine 속성 : 온라인 상태 여부에 대한 정보를 제공 (인터넷 정상 연결 상태라면 true값을 반환)
- userAgent 속성 : 브라우저와 운영체제의 종합 정보를 제공

객체 (Object)
- 객체는 기능과 속성을 가지고 있다.
- 자바스크립트에서도 객체는 속성을 가지고 있고, 객체가 가진 기능들을 메서드라고 한다.
- 객체의 메서드와 속성을 사용하는 기본형은 아래와 같다.
1) 객체.메서드(); -> 객체의 메서드를 실행
2) 객체.속성; -> 객체의 속성값을 가져옴
3) 객체.속성 = 값; -> 객체의 속성값을 변경함

자바스크립트 객체의 종류
- 자바스크립트의 객체는 크게 내장 객체, 브라우저 객체 모델(BOM), 문서 객체 모델로 나눌 수 있다.
1) 내장 객체
- 자바스크립트 엔진에 내장되어 있어 필요한 경우에 생성해서 사용할 수 있다.
- 내장 객체로는 문자(String), 날짜(Date), 배열(Array), 수학(Math) 객체 등이 있다.
- ex) Date 객체에서 오늘 날짜를 알려주는 메서드 getDate() 사용

2) 브라우저 객체 모델
- 브라우저에 계층 구조로 내장되어 있는 객체
- 브라우저 객체로는 window, screen, location, history, navigator 객체 등이 있다.
- 브라우저(window)는 document와 location 객체의 상위 객체이다.
- ex) 자바스크립트를 이용해 현재 사이트에서 타 사이트로 이동하려면 location 객체에 참조 주소(href) 속성을 바꾸면 됨

3) 문서 객체 모델
- HTML 문서 구조를 의미함
- 최상위 객체로 <html>이 있고, 그 하위 객체로 <head>와 <body>가 있다.
- 특정한 요소를 선택해 가져오려면 문서 객체의 구조를 잘 이해해야 한다.
- 문서 객체 모델에서는 HTML의 모든 요소들을 문서 객체로 선택해 자유롭게 속성을 바꿀 수 있고,
  선택한 문서 객체에 원하는 스타일(CSS)을 적용할 수도 있다.

** 하지만 자바스크립트 문서 객체 모델은 IE 8 이하 버전에서는 호환성이 떨어지기 때문에,
   비교적 사용하기 쉬운 제이쿼리 문서 객체 모델을 많이 사용한다.
** 따라서 4장에서는 자바스크립트 문서 객체 모델에 대해 깊히 다루지는 않고, 다양한 객체의 종류와
   각 객체에 내장된 기능(메서드) 및 속성을 이해한다고 생각하자.
